üìã Prerequisites

Before you begin, make sure you have:

  • Node.js 14.0.0 or higher installed
  • npm or yarn package manager
  • Git for version control
  • API credentials for platforms you want to integrate (YouTube, TikTok, Spotify, Vydia)

üöÄ Quick Start

Step 1: Installation

Install ScrollSoul Music Sync via npm:

npm install scrollsoul-music-sync

Or using yarn:

yarn add scrollsoul-music-sync

Step 2: Clone the Repository (for development)

If you want to run the full platform locally:

git clone https://github.com/chaishillomnitech1/scrollsoul-music-sync.git
cd scrollsoul-music-sync
npm install

Step 3: Configuration

Create a .env file in the root directory:

cp .env.example .env

Edit the .env file with your API credentials:

# YouTube API Configuration
YOUTUBE_API_KEY=your_youtube_api_key
YOUTUBE_CLIENT_ID=your_client_id
YOUTUBE_CLIENT_SECRET=your_client_secret

# TikTok API Configuration
TIKTOK_CLIENT_KEY=your_tiktok_client_key
TIKTOK_CLIENT_SECRET=your_tiktok_client_secret
TIKTOK_ACCESS_TOKEN=your_access_token

# Spotify API Configuration
SPOTIFY_CLIENT_ID=your_spotify_client_id
SPOTIFY_CLIENT_SECRET=your_spotify_client_secret

# Vydia API Configuration
VYDIA_API_KEY=your_vydia_api_key
Note: You'll need to obtain API credentials from each platform:
  • YouTube: Google Cloud Console
  • TikTok: TikTok Developers
  • Spotify: Spotify Developer Dashboard
  • Vydia: Contact Vydia for API access

Step 4: Build the Project

npm run build

Step 5: Start the Server

For development:

npm run dev

For production:

npm start
‚úÖ Success! Your server should now be running on http://localhost:3000

üíª Basic Usage

Using as a Library

import { 
  YouTubeClient, 
  TikTokClient, 
  SpotifyClient, 
  VydiaClient 
} from 'scrollsoul-music-sync';

// Initialize clients
const youtube = new YouTubeClient({
  apiKey: process.env.YOUTUBE_API_KEY,
  clientId: process.env.YOUTUBE_CLIENT_ID,
  clientSecret: process.env.YOUTUBE_CLIENT_SECRET
});

const spotify = new SpotifyClient({
  clientId: process.env.SPOTIFY_CLIENT_ID,
  clientSecret: process.env.SPOTIFY_CLIENT_SECRET
});

// Authenticate
await spotify.authenticate();

// Upload to YouTube
const videoId = await youtube.uploadVideo({
  file: './my-music-video.mp4',
  metadata: {
    title: 'My Awesome Song',
    description: 'Official music video',
    tags: ['music', 'original'],
    categoryId: '10' // Music category
  },
  privacyStatus: 'public'
});

// Sync track metadata
const track = await spotify.syncTrackMetadata({
  title: 'My Awesome Song',
  artist: 'My Artist Name',
  album: 'My Album',
  isrc: 'USXX12345678'
});

console.log('YouTube Video ID:', videoId);
console.log('Spotify Track:', track);

üÜò Troubleshooting

Port already in use

Change the PORT in your .env file:

PORT=3001

API authentication errors

  • Verify your API keys are correct
  • Check if access tokens have expired
  • Ensure OAuth redirect URIs are configured correctly

Build errors

# Clear node_modules and reinstall
rm -rf node_modules package-lock.json
npm install

# Rebuild
npm run build
